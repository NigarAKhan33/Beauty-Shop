# DATA  ANALYTICS AND VISUALISATION USING DISNEY+ TITLES DATASET

- The vision for creating this data warehouse utilizing the dataset was to have better understanding of all the facilities provided by the platform and to also have an in depth analysis of the content being offered.
- The goal of this project is to perform an in-depth analysis on the selected dataset and prove a better understanding of the content offered.

# For this project, key stakeholders that are involved are:
 - Disney Plus Directors (who are responsible for the decision of the content being offered on the platform).
-  Movies and show’s creator ( People responsible for the creation of the content being offered ).
 - Actors ( People who work in the movies and shows ).

# The use of SQL server has been done over the regular excel sheets as they :
-  Offer the centralization of the data
- Data integrity and security is improved.
- Scalability can be increased and also the scope of applications broadens.
- Auto recovery and backing of data in server is highly improved.

#The use of visualization tools like the Tableau and SSRS was done to improve the user interface and make the conclusions drawn much more interactive.
#The use of the SSRS has provided us with an opportunity to have much more detailed and business centric reports for the data being worked on.

# tools used #

SQL Server – Data storage, querying, ETL, and schema design

SSRS (SQL Server Reporting Services) – Report creation and distribution

Tableau – Data visualization and interactive dashboard development

Python – Data cleaning, analysis, and support for model evaluation

Excel – Initial exploration and validation

ERD Tools / Diagramming Tools – Dimensional modeling (e.g., Star Schema)
